问题：

在WIN10中,系统自带的防火墙可能会自动删除exe文件，该如何解决这个问题呢？

解决办法：

思路：在win10安全中心中设置文件夹或文件排除项，避免被误杀。

操作步骤：

  1. 打开 Windows Defender


2. 点击左下角齿轮图标


3. 找到“病毒和威胁防护”设置


4. 添加文件所在目录排除项


5. 找到exe，并添加进去


6. 如果还是会自动删除,请把实时保护关了试一下：

打开设置>>更新和安全>>windows安全中心>>病毒和威胁防护>>病毒和威胁防护设置（管理设置）>>关闭实时保护
